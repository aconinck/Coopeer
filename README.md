# Coopeer
App mobile de descoberta e gestão de times esportivos amadores locais. Funciona como um "Airbnb de esportes" — o usuário abre um mapa, vê times perto dele, paga para experimentar um jogo avulso (~$7), e assina uma mensalidade se gostar. Focado no mercado americano inicialmente, começando em Bentonville, AR.
